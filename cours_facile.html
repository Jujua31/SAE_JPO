Journées Portes Ouvertes - Python

Pour créer une variable en python il suffit de mettre le nom de la variable, et la valeur qu’on veut lui donner en séparant les 2 d’un égal. Ex: k=2 Il y a différents types de variables: les int (integer number) qui sont les nombres entiers, les float (floating point number) qui sont les nombres à virgules, les str ( string ) qui sont des chaînes de caractères et qui sont représentés avec des guillemets, et enfin les bool qui sont vrai ou faux, représentés par True et False : Ex: int → 3 float → 2.3 str → “y” bool → True Ces variables peuvent être comparés, additionnés, soustraits… grâce à des symboles. Tout d’abord les signes +, -, *, /, pour additionner, soustraire, multiplier, et diviser. Et on peut aussi les comparer, grâce aux signes < pour inférieur à, > supérieur à, et >= et <= pour superieur ou égale et inférieure ou égale.
