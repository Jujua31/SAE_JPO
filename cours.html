A. Cours Facile

I. Création d'une variable

Pour créer une variable en python il suffit de mettre le nom de la variable, et la valeur qu’on veut lui donner en séparant les 2 d’un égal.

Exemple :

k=2

mot="arbre"

II. Types de variables

Il y a différents types de variables:

  • int (integer number) qui sont les nombres entiers.
  • float (floating point number) qui sont les nombres à virgules,
  • str ( string ) qui sont des chaînes de caractères et qui sont représentés avec des guillemets,
  • bool qui sont vrai ou faux, représentés par True et False.

Exemples :

  • int → 3
  • float → 2.3
  • str → “y”
  • bool → True

III. Comparaison des variables

Ces variables peuvent être comparés, additionnés, soustraits… grâce à des symboles. Tout d’abord les signes +, -, *, /, pour additionner, soustraire, multiplier, et diviser. Et on peut aussi les comparer, grâce aux signes < pour inférieur à, > supérieur à, et >= et <= pour superieur ou égale et inférieure ou égale.

B. Cours niveau moyen

I. Comment définir une fonction, à quoi ça sert ?

Une fonction python est une suite d’instructions à laquelle on peut donner un nom et des paramètres.

Il existe différents types de variables conteneurs :

  • liste, qui est un élement mutable
  • tuple, qui n'est pas un élement mutable

Il existe plusieurs types de “conteneurs” de valeurs. Par exemple, pour contenir des nombres entiers, on peut utiliser une liste ou un tuple. Une liste est dite mutable alors qu’un tuple est non-mutable. Un objet mutable est un objet auquel on peut modifier, enlever ou ajouter les valeurs.

Exemples :

  • ma_liste = [0,1]
  • ma_liste.append(2)
  • ma_liste → [0,1,2]
  • mon_tuple = (0,1)
  • mon_tuple.append(2) → provoque une erreur

II. Accès aux valeurs

Pour accéder à une valeur dans un tuple ou une liste, on utilise l’indexation. Chaque élément est associé à un “rang”. Le premier élément est associé au rang 0, le deuxième au rang 1, etc…

Exemples :

  • ma_liste = [1,2,3]
  • ma_liste[0] → renvoie 1
  • ma_liste[2] → renvoie 3

C. Cours niveau difficile

I. Itérer grâce à une boucle

Pour parcourir les éléments d’une liste, on peut utiliser la boucle for avec soit un parcours par élément donc en prenant directement chaque élément ou un parcours par indice en prenant les positions des éléments dans la liste, qui fonctionne avec range, dans laquelle on entre la taille de la liste.

Exemples :

“for elt in liste” pour le parcours par élément

“for i in range(len(liste))” pour le parcours par élément.

II.

On peut aussi faire un parcours par indice avec la boucle while, en imposant une condition qui, tant qu’elle est respectée, la boucle est lancée.

Exemple :

“i=0; while i” et en ajoutant 1 à i à chaque tour de boucle.
